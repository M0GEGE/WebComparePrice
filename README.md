Веб-приложения для сравнения цен Compare All

Курсовая работа выполнена с использованием:

  HTML;
  
  CSS;
  
  Javascript.
  
Для Backend части использовались:

  NodeJS + Express.
  
Реализованный функционал:

  создание товаров;
  
  редактирование созданного товара;
  
  удаление товара;
  
  просмотр всех существующих товаров;
  
  сравнение существующих товаров по цене, вывод результата сравнений от min к max.
  
База данных MySQL 8.0

📖 Описание проекта
Compare All — это полнофункциональное веб-приложение для управления товарами и сравнения их цен. Проект разработан в рамках курсовой работы и демонстрирует навыки создания клиент-серверных приложений с использованием современного стека технологий.

✨ Основные возможности
Управление товарами: создание, просмотр, редактирование и удаление товаров через удобный интерфейс.

Визуальное представление: товары отображаются в виде цветных карточек со случайными градиентами.

Сравнение цен: одним нажатием кнопки все товары сортируются по возрастанию цены.

Наглядный результат: минимальная цена выделяется зелёным, максимальная — красным с соответствующими метками (MIN/MAX).

Горизонтальная прокрутка: результаты сравнения удобно просматривать при большом количестве товаров.

Адаптивный дизайн: единый стиль на всех страницах, согласованная типографика и цветовая схема.

📌 Особенности реализации

Frontend

Все стили вынесены в отдельные CSS-файлы и переиспользуются на разных страницах.

Карточки товаров генерируются динамически через JavaScript.

Модальные окна для добавления и редактирования товаров без перезагрузки страницы.

Горизонтальная прокрутка для длинных списков результатов.

Backend

REST API с эндпоинтами: GET, POST, PUT, DELETE /products.

Все ответы сервера возвращаются в формате JSON.

База данных
MySQL 8.0 с таблицей products (id, name, price).
